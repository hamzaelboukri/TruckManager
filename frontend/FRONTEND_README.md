# TruckManager Frontend

## Structure du Projet

src/
 components/      # Composants réutilisables
    common/      # Composants communs (ProtectedRoute, RoleGuard)
    auth/        # Composants d'authentification
    trucks/      # Composants des camions
    trailers/    # Composants des remorques
    tires/       # Composants des pneus
    routes/      # Composants des routes
    maintenance/ # Composants de maintenance
    dashboard/   # Composants du tableau de bord
 pages/          # Pages de l'application
 services/       # Services API
 store/          # Redux store & slices
 hooks/          # Custom hooks
 utils/          # Utilitaires
 types/          # Types TypeScript
 layouts/        # Layouts de l'application
 contexts/       # React contexts
 assets/         # Images, icons, etc.

## Technologies
- React 18
- TypeScript
- Vite
- TailwindCSS
- Redux Toolkit
- React Router v6
- Axios
- React Hot Toast
- Lucide Icons
